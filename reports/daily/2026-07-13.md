# Agent 技术日报 · 2026-07-13

## GitHub projects

### [langflow](https://github.com/langflow-ai/langflow)
- 支持通过可视化界面构建和编排多智能体（multi-agent）系统，集成MCP协议实现模块间通信。
- 内置RAG（检索增强生成）能力，可对接向量数据库与文档源，增强AI工作流的知识调用。
- 提供tool calling机制，允许在流程中无缝调用外部API与自定义工具，扩展智能体功能边界。

### [Dify](https://github.com/langgenius/dify)
- 支持 MCP 协议集成，实现模型上下文感知的智能决策
- 内置 RAG 管道，无缝衔接知识库检索与生成式工作流
- 提供 tool calling 与 multi-agent 编排能力，构建复杂协作式智能体

### [AI工具的系统提示、内部工具与模型集合](https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools)
- 涵盖MCP集成、RAG增强搜索、工具调用(tool calling)及多智能体(multi-agent)架构。
- 包括Cursor、Claude Code、Devin AI、Lovable等主流AI编码助手及v0、Dia等设计工具的系统提示。
- 开放系统提示、内部工具与模型细节，促进透明度与可复现性。

### [langchain](https://github.com/langchain-ai/langchain)
- 支持 MCP（Model Context Protocol）实现标准化上下文管理与外部系统交互
- 集成 RAG（检索增强生成）以从外部知识库检索相关文档提升生成质量
- 提供 tool calling 和 multi-agent 编排能力，支持复杂任务分解与协作执行

### [claude-code](https://github.com/anthropics/claude-code)
- 通过自然语言指令执行日常编码任务，支持 MCP 协议扩展能力
- 利用 RAG 技术理解代码库上下文，提供精准的复杂代码解释
- 集成 tool calling 实现 Git 工作流自动化，支持 multi-agent 协作模式

### [agency-agents](https://github.com/msitarzewski/agency-agents)
- 利用MCP协议整合各Agent服务，实现工具调用与资源访问的无缝对接。
- 通过RAG技术为Agent提供实时知识库支持，确保回应精准且上下文相关。
- 采用Multi-Agent协作架构，各专业Agent并行处理任务并动态协调，提升整体效率。
