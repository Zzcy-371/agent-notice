## GitHub projects

### [agency-agents](https://github.com/msitarzewski/agency-agents)
- 通过MCP协议集成多种专业agent，支持RAG知识库与tool calling实现上下文感知和动态工具调用。
- 每个agent具备独立人格与工作流，覆盖前端开发、社区运营、创意注入和现实检验等场景。
- 支持multi-agent协作编排，agent间可通信、委派任务并协同完成复杂交付。


## Research and engineering

### [Show HN: Deep Skill Finder – 使用真实执行基准寻找智能体技能](https://www.meyo.life/skill)
- 通过MCP和RAG增强，基于真实执行基准评估智能体技能
- 支持tool calling和multi-agent协作，精准匹配技能需求
- 提供可复现的性能指标，帮助开发者优化智能体任务分配

### [Harness Handbook – Making Agent Harnesses Understandable, Auditable and Editable](https://ruhan-wang.github.io/Harness-Handbook)
- 讨论了如何设计可理解、可审计、可编辑的 agent harness，支持 MCP、RAG、tool calling 和 multi-agent 等能力。
- 提出 harness 应作为透明层，记录所有 agent 与外部工具的交互，便于调试和审计。
- 强调 harness 的可编辑性允许动态调整 agent 行为，例如修改 RAG 检索参数或切换 multi-agent 协作策略。

### [Chrysalis Zero Trust Agent Registry](https://baursoftware.com/zero-trust/chrysalis)
- Chrysalis 利用 MCP 协议在零信任架构中实现安全代理注册与动态身份验证。
- 通过 RAG 技术检索策略数据库，Chrysalis 自动管理代理权限并与 tool calling 集成。
- 支持 multi-agent 协作，Chrysalis 确保代理间通信在零信任环境下加密且可审计。
