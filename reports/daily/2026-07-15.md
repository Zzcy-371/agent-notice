# Agent 技术日报 · 2026-07-15

## GitHub projects

### [superpowers](https://github.com/obra/superpowers)
- 超级能力（Superpowers）是一个结合了多智能体（multi-agent）编排与MCP（模型上下文协议）的智能体技能框架，用于构建协作式AI系统。
- 通过RAG（检索增强生成）与tool calling机制，该框架可集成外部知识库与工具，增强智能体的上下文理解与执行能力。
- 提供一套软件开发方法论，指导开发者高效设计、部署和迭代基于智能体的应用程序，确保实际落地效果。

### [ECC: 智能体能力优化系统](https://github.com/affaan-m/ECC)
- 集成MCP协议实现工具调用（tool calling），通过标准化接口增强Claude Code、Codex等智能体对外部系统的交互能力。
- 利用RAG技术优化记忆与检索机制，结合多智能体（multi-agent）协作架构提升复杂任务的执行效率。
- 提供安全沙箱与技能编排功能，支持Cursor、Opencode等平台的性能调优及研究优先开发模式。

### [hermes-agent](https://github.com/NousResearch/hermes-agent)
- 支持MCP协议，无缝集成多种工具调用（tool calling）能力，实现高效自动化任务编排。
- 内置RAG检索增强生成，结合多智能体（multi-agent）协作，动态扩展知识库与推理能力。
- 提供渐进式学习接口，根据交互反馈自适应性优化Agent行为，真正实现与用户共同成长的智能体。

### [opencode](https://github.com/anomalyco/opencode)
- opencode 作为一个 MCP（模型上下文协议）兼容的编码代理，支持工具调用（tool calling）实现代码编辑、文件操作和终端命令执行。
- 它利用 RAG（检索增强生成）技术从项目文档或代码库中检索上下文，提升代码生成和理解的准确性。
- opencode 通过多智能体（multi-agent）协作架构，协调不同专用代理完成复杂编码任务，如调试、重构和测试。

### [skills](https://github.com/anthropics/skills)
- 支持MCP（Model Context Protocol）作为工具调用标准，提供模块化技能封装与上下文管理。
- 集成RAG（Retrieval-Augmented Generation）能力，允许Agent通过检索外部知识库增强回答准确性。
- 内置Multi-Agent协作框架，支持技能间的动态组合与tool calling编排，实现复杂任务自动化。

### [dify](https://github.com/langgenius/dify)
- 支持MCP协议集成，通过标准接口连接外部工具与数据源。
- 内置RAG引擎，实现知识库检索增强生成，提升回答准确性。
- 提供tool calling和multi-agent编排，支持复杂工作流自动化。
