### [awesome-design-md](https://github.com/VoltAgent/awesome-design-md)
- 整合知名品牌设计系统的 DESIGN.md 文件，通过 MCP 协议让 coding agent 调用 tool calling 自动生成匹配 UI。
- 支持 RAG 检索设计规范，结合 multi-agent 协作实现跨组件样式的精准统一。
- 项目内置 tool calling 接口，使 agents 能直接读取设计 tokens 并同步生成前端代码。

### [codex](https://github.com/openai/codex)
- 轻量级终端编码代理，支持 MCP 协议扩展工具调用能力。
- 集成 RAG 检索增强生成，可索引本地代码库进行上下文感知的补全。
- 提供 multi-agent 协同框架，允许专门子代理协作处理复杂任务。

### [TradingAgents](https://github.com/TauricResearch/TradingAgents)
- 基于MCP协议构建多智能体协作系统，支持RAG机制从历史交易数据中检索策略，触发tool calling执行买卖操作。
- 通过multi-agent框架实现交易决策、风险管理与市场分析并行处理，各代理通过MCP交换信号。
- 利用LLM进行自然语言指令解析，结合RAG动态调用技术指标计算工具与外部数据源。
