# Agent 技术日报 · 2026-07-20

## GitHub projects

### [agent-skills](https://github.com/addyosmani/agent-skills)
- 提供生产级MCP服务器实现，支持AI编码代理与外部工具的高效交互。
- 集成RAG能力，增强代理对上下文文档的检索与生成准确性。
- 支持tool calling和multi-agent协作，实现复杂任务的自动化分配与执行。

### [Prompt-Engineering-Guide](https://github.com/dair-ai/Prompt-Engineering-Guide)
- 涵盖提示工程、上下文工程、RAG 和 AI 智能体的指南、论文、课程与实践资源
- 支持 MCP 协议集成，便于 AI 智能体调用外部工具和模型上下文
- 深入讲解 tool calling 与 multi-agent 协作策略，提升复杂任务执行效率

### [MinerU](https://github.com/opendatalab/MinerU)
- MinerU 将 PDF、Office 等复杂文档转换为 LLM 就绪的 markdown/JSON，支持 MCP 集成与大模型推理优化。
- 通过 RAG 技术提取关键信息，结合 tool calling 与 multi-agent 协同，实现文档驱动的智能自动化。
- 为 Agentic workflows 提供标准化数据接口，助力多智能体系统高效解析与决策。

### [paperclip](https://github.com/paperclipai/paperclip)
- 支持通过 MCP 集成多种 AI 代理，实现统一管理。
- 内置 RAG 管道，可高效检索企业内部知识库。
- 提供 tool calling 和 multi-agent 协作，自动化复杂工作流。
