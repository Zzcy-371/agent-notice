# Agent 技术日报 · 2026-07-19

## GitHub projects

### [autoresearch](https://github.com/karpathy/autoresearch)
- 利用MCP协议协调多个AI agent，实现单GPU nanochat训练研究的全自动化流程。
- 通过RAG技术动态检索最新文献，实时优化训练策略与模型架构。
- 结合tool calling与multi-agent协作，自动完成数据采集、超参调优与结果验证。

### [claude-mem](https://github.com/thedotmack/claude-mem)
- 通过 MCP 连接和 RAG 压缩技术，自动捕获代理会话中的工具调用、多智能体交互及对话历史，生成持久化上下文记忆。
- 利用 AI 对跨会话的 agent 行为进行智能摘要，并在未来会话中注入相关上下文，支持 Claude Code、OpenClaw、Codex 等工具的 tool calling 连续性。
- 为多智能体协作场景提供统一记忆层，确保不同代理在工具调用和决策时共享历史状态，减少重复上下文传递。

### [RAGFlow](https://github.com/infiniflow/ragflow)
- RAGFlow 是一款领先的开源检索增强生成引擎，融合 MCP、RAG 与 tool calling，为大语言模型打造优质上下文层。
- 支持多智能体协作，通过 agent 编排实现复杂任务的自动拆解与执行。
- 内置强大 RAG 管道，结合 MCP 协议与工具调用，提升检索精准度与生成连贯性。

### [lobehub](https://github.com/lobehub/lobehub)
- LobeHub 作为首席代理操作员，通过 MCP 协议协调 AI 团队，支持 RAG 增强的知识检索与 tool calling 工具调用。
- 采用多 agent 架构，实现 7×24 小时自动化调度与任务分配，提升团队协作效率。
- 提供全面的报告机制，实时监控代理状态与操作结果，优化整体工作流性能。
